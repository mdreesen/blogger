# blogger

## Description
This site is for bloggers to post their blog links to.

Link: https://murmuring-reaches-11111.herokuapp.com/

## Running the app
npm start -> This starts running the express backend server

## Used in the project
 - dotenv
 - sequelize
 - mysql2
 - bcrypt
 - express-session
 - connect-session-sequelize
 - express-handlebars
